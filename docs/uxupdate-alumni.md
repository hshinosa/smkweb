# Perencanaan Update UX Modul Jejak Alumni

Dokumen ini berisi rencana peningkatan User Experience (Basic/Must Have) untuk modul Jejak Alumni.

## 1. Filter Profesi & Tahun Lulus (Public)
**Masalah:**
Siswa yang ingin motivasi spesifik (misal: ingin masuk Akmil) sulit menemukan alumni dengan jalur tersebut di antara tumpukan testimoni.

**Solusi:**
Filter kombinasi di halaman Alumni.
- **Filter:** Tahun Lulus (Angkatan) dan Bidang Pekerjaan/Kuliah (TNI/Polri, Kedokteran, Engineering, Bisnis).

## 2. Form "Kirim Kisahmu" (Public/Engagement)
**Masalah:**
Admin kesulitan mengumpulkan data alumni secara manual (menghubungi satu per satu).

**Solusi:**
Tombol "Saya Alumni - Kirim Testimoni" di halaman publik.
- **Mekanisme:** Form terbuka untuk alumni mengisi data (Nama, Tahun, Profesi, Pesan, Foto).
- **Keamanan:** Data masuk sebagai status "Pending" di Admin. Admin harus review & approve sebelum tampil di web.

## 3. Badge "Verifikasi" (Admin)
**Masalah:**
Validitas testimoni perlu dijaga.

**Solusi:**
Admin bisa menandai alumni yang sudah diverifikasi (valid). Tampil centang biru atau label "Verified Alumni" di kartu publik.
