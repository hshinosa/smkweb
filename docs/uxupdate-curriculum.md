# Perencanaan Update UX Modul Kurikulum (Deep Learning Focus)

Dokumen ini berisi rencana peningkatan User Experience (Basic/Must Have) untuk modul Kurikulum yang berfokus pada konsep Deep Learning.

## 1. Sticky Table of Contents (Public)
**Masalah:**
Halaman kurikulum sangat panjang (Long-scrolling) dengan banyak sub-bab teoritis (Definisi, Prinsip 3M, Siklus Belajar). Pengunjung sulit menavigasi atau mencari bagian tertentu.

**Solusi:**
Menu Navigasi Cepat (Sticky Sidebar/Header).
- **Isi:** "Masalah & Tantangan", "Apa itu Deep Learning", "Prinsip 3M", "Siklus Belajar", "Profil Pelajar".
- **Interaksi:** Klik menu -> Scroll otomatis ke section terkait.

## 2. Interactive Learning Cycle (Public)
**Masalah:**
Konsep "Siklus Belajar" atau "4 Olah" disajikan dalam grid kartu teks statis yang membosankan dan memakan tempat vertikal.

**Solusi:**
Visualisasi Interaktif (Tabs atau Circular Diagram).
- **Tampilan:** Diagram siklus di tengah. Klik fase "Olah Pikir" -> Muncul penjelasan detail di sampingnya.
- **Benefit:** Membuat konsep abstrak lebih mudah dipahami dan lebih *engaging*.

## 3. Download Center Integration (Public)
**Masalah:**
Halaman saat ini murni berisi filosofi kurikulum. Orang tua/siswa yang mencari dokumen praktis (Kalender Akademik, Jadwal Pelajaran, Dokumen Kurikulum Merdeka) tidak menemukannya di sini.

**Solusi:**
Section "Dokumen Akademik" di bagian akhir.
- **Fitur:** List download file PDF (Kalender, Silabus, RPP/Modul Ajar).
- **Integrasi:** Bisa mengambil data dari modul *Academic Calendar* atau upload manual di settings kurikulum.
