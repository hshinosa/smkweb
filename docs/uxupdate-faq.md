# Perencanaan Update UX Modul FAQ

Dokumen ini berisi rencana peningkatan User Experience (Basic/Must Have) untuk modul FAQ (Tanya Jawab).

## 1. Live Search (Public)
**Masalah:**
Daftar FAQ yang panjang membuat pengguna malas membaca satu per satu untuk menemukan jawaban masalah mereka.

**Solusi:**
Kotak Pencarian "Cari Pertanyaan...".
- **Mekanisme:** Filter instan saat mengetik. Ketik "SPP", langsung muncul semua Q&A yang mengandung kata SPP.

## 2. Accordion Grouping (Public)
**Masalah:**
Pertanyaan PPDB bercampur dengan pertanyaan Sarana Prasarana.

**Solusi:**
Kelompokkan FAQ dalam Accordion per Topik.
- **Topik:** PPDB, Akademik, Keuangan, Fasilitas.
- **Interaksi:** Klik topik -> Muncul daftar pertanyaan topik tersebut.

## 3. Feedback Loop (Public)
**Masalah:**
Admin tidak tahu apakah jawaban di FAQ sudah cukup membantu atau membingungkan.

**Solusi:**
Tombol "Apakah jawaban ini membantu? (Ya/Tidak)" di bawah setiap jawaban.
- **Admin:** Melihat statistik "helpful" untuk merevisi jawaban yang banyak dapat vote "Tidak".
