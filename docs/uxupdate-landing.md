# Perencanaan Update UX Modul Landing Page (Beranda)

Dokumen ini berisi rencana peningkatan User Experience (Basic/Must Have) untuk halaman utama website.

## 1. Quick Links / Akses Cepat (Public - Mobile First)
**Masalah:**
Di layar HP, menu navigasi tersembunyi dalam "Hamburger Menu". Pengguna butuh akses instan ke info vital tanpa klik menu.

**Solusi:**
Bar "Akses Cepat" di bawah banner utama.
- **Isi:** Icon besar untuk "PPDB", "Cek Kelulusan", "E-Learning", "Kontak".
- **Sticky:** Bisa dibuat melayang (sticky) di bagian bawah layar HP.

## 2. Emergency/Announcement Banner (Public)
**Masalah:**
Info mendesak (misal: "Sekolah Diliburkan", "Website Sedang Maintenance") sulit tersampaikan jika harus membuat postingan berita biasa.

**Solusi:**
Global Alert Banner di paling atas website.
- **Admin:** Toggle On/Off + Input Teks + Pilihan Warna (Merah/Kuning/Biru).

## 3. Live Preview Content (Admin)
**Masalah:**
Mengedit teks sambutan atau slogan di form admin terasa "buta" karena tidak tahu hasil akhirnya.

**Solusi:**
Preview Pane di halaman edit.
- **Tampilan:** Sisi kiri form input, sisi kanan render tampilan (mirip blog editor modern).
