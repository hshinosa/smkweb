# Perencanaan Update UX Modul Program Studi (Jurusan)

Dokumen ini berisi rencana peningkatan User Experience (Basic/Must Have) untuk halaman Program Studi (MIPA/IPS/Bahasa).

## 1. Visualisasi Mata Pelajaran Peminatan (Public)
**Masalah:**
Calon siswa bingung "Bedanya pelajaran di IPA dan IPS apa saja?". Penjelasan teks paragraf sering dilewatkan.

**Solusi:**
Card/Grid "Mapel Khas".
- **Tampilan:** Icon + Nama Mapel (Misal: Biologi, Kimia, Fisika untuk MIPA).
- **Interaksi:** Hover untuk lihat deskripsi singkat mapel.

## 2. Galeri Kegiatan Spesifik (Public)
**Masalah:**
Halaman prodi terasa kering hanya berisi teks kurikulum. Siswa ingin lihat "serunya" masuk jurusan itu.

**Solusi:**
Slider Foto Kegiatan Jurusan.
- **Konten:** Foto praktikum di Lab (MIPA), Debat/Kunjungan (IPS), Lab Bahasa/Teater (Bahasa).

## 3. Integrasi Alumni Jurusan (Public)
**Masalah:**
Siswa butuh bukti sukses (role model) dari jurusan tersebut.

**Solusi:**
Widget "Alumni Sukses [Jurusan]".
- **Fitur:** Menampilkan 3 kartu profil alumni yang berasal dari jurusan terkait (diambil dari modul Alumni dengan filter tags).
