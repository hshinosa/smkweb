# Perencanaan Update UX Modul Program Sekolah

Dokumen ini berisi rencana peningkatan User Experience (Basic/Must Have) untuk modul Program Sekolah.

## 1. Visualisasi Status Program (Public)
**Masalah:**
Pengunjung tidak tahu apakah program "Penghijauan Sekolah" itu masih berjalan, sudah selesai tahun lalu, atau baru rencana.

**Solusi:**
Badge Status yang Jelas.
- **Label:** "Aktif", "Selesai", "Mendatang".
- **Warna:** Hijau (Aktif), Abu-abu (Selesai), Biru (Mendatang).

## 2. Downloadable Resources (Public)
**Masalah:**
Program sekolah seringkali memiliki dokumen panduan, juknis, atau formulir pendaftaran yang perlu diakses siswa/guru.

**Solusi:**
Bagian "Dokumen Pendukung" di detail program.
- **Fitur:** List file PDF/Docx yang bisa diunduh langsung.

## 3. Tab Navigasi Kategori (Public)
**Masalah:**
Program sekolah beragam (Akademik, Lingkungan, Karakter). List vertikal panjang membosankan.

**Solusi:**
Navigasi Tab Horizontal.
- **Kategori:** Program Unggulan, Adiwiyata, Literasi, Kesiswaan.
