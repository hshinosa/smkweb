# Perencanaan Update UX Modul Site Settings

Dokumen ini berisi rencana peningkatan User Experience (Basic/Must Have) untuk pengaturan situs.

## 1. Social Media Link Validator (Admin)
**Masalah:**
Admin sering typo saat input URL Instagram/Facebook, menyebabkan link di footer website mati (404).

**Solusi:**
Tombol "Test Link".
- **Aksi:** Membuka URL yang diinput di tab baru untuk verifikasi manual sebelum simpan.

## 2. Maintenance Mode (Admin)
**Masalah:**
Saat update data besar-besaran, website terlihat berantakan bagi pengunjung.

**Solusi:**
Toggle "Mode Perbaikan".
- **Efek:** Halaman publik dialihkan ke splash screen "Sedang Maintenance", tapi Admin tetap bisa akses.

## 3. SEO Preview Global (Admin)
**Masalah:**
Admin tidak tahu bagaimana tampilan website jika dibagikan di WhatsApp/Facebook.

**Solusi:**
Preview OG Image & Meta Tags.
- **Tampilan:** Card simulasi tampilan link preview WhatsApp.
